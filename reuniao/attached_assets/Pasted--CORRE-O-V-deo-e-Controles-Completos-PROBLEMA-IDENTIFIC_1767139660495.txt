## 🐛 SE O VÍDEO AINDA NÃO APARECER

### Problema 1: Permissões do Browser
**Solução:**
1. Clique no ícone de câmera/microfone na barra de URL
2. Permita acesso à câmera e microfone
3. Recarregue a página

### Problema 2: Dispositivos não detectados
**Solução:**
1. Abra DevTools (F12)
2. Console → Digite:
```javascript
navigator.mediaDevices.enumerateDevices().then(console.log)
```
3. Verifique se aparecem dispositivos de vídeo/áudio

### Problema 3: Erro no Console
**Verificar:**
```bash
# No console do browser (F12):
- Procure por erros em vermelho
- Procure por "attach" ou "detach" errors
- Procure por "permission denied"
```

**Se ver erro de permissão:**
- Abra em **HTTPS** (Replit já usa HTTPS)
- Chrome/Firefox bloqueiam câmera em HTTP

## 🎨 RECURSOS DA INTERFACE

### Header (Topo)
- Logo do tenant (se configurado)
- Nome da empresa
- Contador de participantes
- Código da reunião (opcional)

### Grid de Vídeos (Centro)
- Vídeo de cada participante
- Ou avatar com inicial do nome
- Nome do participante no canto inferior
- Indicador de microfone mudo (ícone vermelho)
- Layout responsivo:
  - 1 pessoa: 1 coluna (tela cheia)
  - 2 pessoas: 2 colunas
  - 3-4: Grid 2x2
  - 5-9: Grid 3x3
  - 10+: Grid 4x4

### Controles (Inferior)
- **Microfone**: Azul = ligado, Vermelho = mudo
- **Câmera**: Azul = ligada, Vermelho = desligada
- **Compartilhar Tela**: Inicia/para compartilhamento
- **Chat**: Abre sidebar direita com chat
- **Participantes**: Abre sidebar com lista
- **Sair**: Vermelho, encerra reunião

### Sidebars
- **Chat**: 
  - Abre pela direita
  - Área para mensagens (em desenvolvimento)
  - Botão X para fechar

- **Participantes**:
  - Lista todos os participantes
  - Avatar circular com inicial
  - Nome + função (role)
  - Indicador de mic mudo
  - Botão X para fechar

## 💡 DICAS

1. **Vídeo não aparece?**
   - Verifique permissões do browser
   - Teste em outro browser
   - Verifique se câmera funciona em outros apps

2. **Performance ruim?**
   - Feche outras abas
   - Use navegador atualizado
   - Verifique conexão de internet

3. **Quer customizar cores?**
   - Edite `config.colors` no banco de dados
   - Cores são aplicadas automaticamente

4. **Compartilhamento de tela não funciona?**
   - Precisa estar em HTTPS (Replit já tem)
   - Alguns browsers bloqueiam em iframe
   - Teste em janela separada
