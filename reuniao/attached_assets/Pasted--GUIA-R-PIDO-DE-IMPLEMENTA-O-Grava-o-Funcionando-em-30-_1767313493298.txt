## 7️⃣ ADICIONAR BOTÃO DE GRAVAR (5 min)

Edite `client/src/components/Meeting100ms.tsx`:

**Adicionar imports:**
```typescript
import { Circle } from "lucide-react";
```

**Adicionar estados:**
```typescript
const [isRecording, setIsRecording] = useState(false);
const [recordingId, setRecordingId] = useState<string | null>(null);
```

**Adicionar função:**
```typescript
const handleToggleRecording = async () => {
  try {
    const roomId = window.location.pathname.split('/').pop();
    
    if (!isRecording) {
      // Iniciar
      const response = await fetch('/api/100ms/recording/start', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ roomId }),
      });

      if (!response.ok) throw new Error('Erro ao iniciar');

      const data = await response.json();
      setRecordingId(data.recordingId);
      setIsRecording(true);
      alert('✅ Gravação iniciada!');
      
    } else {
      // Parar
      const response = await fetch('/api/100ms/recording/stop', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ roomId }),
      });

      if (!response.ok) throw new Error('Erro ao parar');

      setIsRecording(false);
      setRecordingId(null);
      alert('⏸️ Gravação parada! Será processada em 5-10 minutos.');
    }
  } catch (err: any) {
    alert('❌ ' + err.message);
  }
};
```

**Adicionar botão nos controles (após botão de tela):**
```typescript
{/* Botão de Gravar (só para host) */}
{localPeer?.roleName === 'host' && (
  <Button
    onClick={handleToggleRecording}
    variant={isRecording ? "destructive" : "outline"}
    size="lg"
    className="flex items-center gap-2"
  >
    <Circle 
      className={`w-5 h-5 ${isRecording ? 'animate-pulse fill-red-500' : ''}`} 
    />
    <span className="hidden sm:inline">
      {isRecording ? 'Parar' : 'Gravar'}
    </span>
  </Button>
)}
```

## 🐛 TROUBLESHOOTING RÁPIDO

### Erro: "Token não fornecido"
**Causa:** URL sem `?token=...`
**Solução:** Verificar endpoint de start que está incluindo token na URL

### Erro: "Role não encontrada"
**Causa:** Role `beam_recorder` não existe
**Solução:** Criar role no dashboard 100ms

### Vídeo não aparece na página /recording
**Causa:** Tracks não estão sendo anexados
**Solução:** Verificar logs no console, pode precisar aguardar mais tempo

### Gravação com tela preta
**Causa:** Bot indo para URL errada
**Solução:** Verificar `PUBLIC_URL` no .env

### Erro 401 na API 100ms
**Causa:** `HMS_MANAGEMENT_TOKEN` inválido
**Solução:** Gerar novo token no dashboard
