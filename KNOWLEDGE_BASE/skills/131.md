**Objective:** Deliver interfaces that feel expensive, trustworthy, and fluid.

## 1. The "Visual First" Mandate ðŸ“¸
*   **Never Code Blind:** Before writing a React component, visualize:
    *   What is the hierarchy? (Big bold headers).
    *   Where is the whitespace? (Generous padding).
    *   What is the accent color? (Consistent palettes).

## 2. The "Premium Feel" Checklist âœ¨
*   **Typography:** Use modern sans-serifs (Inter, Geist, SF Pro). Tight letter-spacing on headers, relaxed on body.
*   **Glassmorphism:** Subtle background blurs (`backdrop-filter: blur(10px)`) for overlays and sticky headers.
*   **Shadows:** Deep, multi-layered shadows for depth. (e.g., `box-shadow: 0 4px 6px -1px rgba(0,0,0,0.1), 0 2px 4px -1px rgba(0,0,0,0.06)`).
*   **Borders:** Subtle 1px borders with low opacity (`border-white/10`) for structure without heaviness.

## 3. Motion & Micro-Interactions âš¡
*   **Rule:** static is boring.
*   **Hover:** Buttons should lift (`transform: translateY(-1px)`).
*   **Click:** Buttons should press (`scale(0.98)`).
*   **Loading:** Skeleton screens (shimmer) instead of spinners.
*   **Transitions:** All state changes must have `transition-all duration-200`.

## 4. Mobile-First Obedience ðŸ“±
*   **The Thumb Zone:** Critical actions must be reachable with one thumb.
*   **Touch Targets:** Minimum 44x44px for all clickable elements.
*   **No Horizontal Scroll:** content must NEVER overflow horizontally on mobile.
