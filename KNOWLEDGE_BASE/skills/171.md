## 🤖 PROTOCOLO n8n
1. **Conexão:** Localizado em http://localhost:5678.
2. **Monitoramento:** O Sentinel Triad deve verificar se o container 'n8n-standalone' está Up.
3. **Ação:** O Antigravity pode agora delegar tarefas de longa duração e automações de redes sociais para o n8n via API.
4. **Endpoint:** Utilizar N8N_API_KEY para autenticação nas requisições do Overseer.
